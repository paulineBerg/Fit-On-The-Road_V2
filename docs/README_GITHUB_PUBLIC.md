## Overview

Fit On The Road is a high-performance React + Vite website focused on:

-   Corporate QVT & Performance programs (B2B)
-   Individual coaching (B2C)
-   Local SEO strategy (Versailles & Saint-Quentin-en-Yvelines)

This repository contains the full production-ready frontend application.

------------------------------------------------------------------------

## Tech Stack

-   React 18 + TypeScript
-   Vite
-   Material UI (dark theme)
-   Tailwind (utility layer)
-   React Router
-   Google Analytics 4 (consent-based)
-   Tarteaucitron (Consent Mode v2)

## Development

Install dependencies:
npm ci

Run locally:
npm run dev

Build production:
npm run build

Build deployment:
npm run deploy:bundle

------------------------------------------------------------------------

## Deployment

-   Static build via Vite
-   Apache hosting
